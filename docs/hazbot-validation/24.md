## Factor Variables

- **ranSimulation** — At least one "SimulationStarted" event was recorded.
- **setTerrainType** — There is at least one "SimulationStarted" event for which the terrain type was set distinct from the default value for any zone.
  - Details: Terrain type is set per zone, in the zones data. Default values (see the "SIMINIT" sheet) = "Plains" (zone 1), "Plains" (zone 2). <i> means taking the zone index i (the index for the zones data, which is, or must be, an array), 0-based. So i=0 means zone 1 and i=1 means zone 2. If true, then necessarily ranSimulation=true.
- **setVegetation** — There is at least one "SimulationStarted" event for which the vegetation type was set distinct from the default value for any zone.
  - Details: Vegetation is set per zone, in the zones data. For the default values, read the "SIMINIT" sheet. If true, then necessarily ranSimulation=true.
- **setDroughtLevel** — There is at least one "SimulationStarted" event for which the drought level was set distinct from the default value for any zone.
  - Details: Drought level is set per zone, in the zones data. For the default values, read the "SIMINIT" sheet. If true, then necessarily ranSimulation=true.
- **setAnyZoneVar** — setTerrainType OR setVegetation OR setDroughtLevel
- **uniqueWindValuesUsed** — The set of unique wind values used for all "SimulationStarted" events.
  - Details: Wind is set globally (for all zones). Default values = 0 (magnitude), 0 (direction) (see the "SIMINIT" sheet). If the magnitude is 0, then the direction has no effect and must be ignored. So set the direction to null, if the magnitude is 0. Here, the "magnitude" means the wind speed as displayed in the simulation (like "10" in "10 MPH"). In the log data, the magnitude data entails two fields "wind.speed" and "wind.scaleFactor". The "magnitude" is computed as "wind.speed" / "wind.scaleFactor". Collect all unique wind values {magnitude, direction} (with direction = null if magnitude = 0).
- **uniqueNonZeroWindValuesUsed** — uniqueWindValuesUsed with no wind (magnitude = 0) removed.
