- **Fireline** — Sim prop for whether any firelines were set up for the simulation run.

  - Details: A "Helitack" event is associated with a simulation run if it is triggered during a simulation run, but a Helitack event can also occur outside a simulation run. - **DefaultVegetations** — Sim prop for whether all variables were held at default values.
  - Details: The values of vegetation for all three zones must be equal to their default values (see the "SIMINIT" sheet) for this sim prop to be true.
- **SevereDroughts** — Sim prop for whether the drought levels for all zones were set to severe.
  - Details: The values of droughtLevel for all three zones must be set to "Severe Drought".
